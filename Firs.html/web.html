    public class CompareIntegers {
      public static void main(String[] args) {
          int num1 = 5;
          int num2 = 10;
  
          for (int i = 0; i < 5; i++) {
              // Compare the values in the loop
              if (num1 < num2) {
                  System.out.println("num1 is less than num2");
              } else if (num1 > num2) {
                  System.out.println("num1 is greater than num2");
              } else {
                  System.out.println("num1 is equal to num2");
              }
              
              // Increment or modify the values for the next iteration
              num1++;
              num2--;
          }
      }
  }
  In this example, we have two integers num1 and num2 with initial values 5 and 10, respectively. Inside the loop, we compare the values using an if statement. The loop will execute five times, comparing the values in each iteration and printing the appropriate message based on the comparison result.
  
  You can modify the values of num1 and num2 or change the loop condition to suit your requirements.
